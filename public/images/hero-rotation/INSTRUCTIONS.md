# Инструкции за добавяне на снимките

## Стъпки:
1. **Запази всяка снимка** с правилното име в папката `public/images/hero-rotation/`
2. **Именувай снимките така:**
   - Първата снимка → `image1.jpg`
   - Втората снимка → `image2.jpg`
   - Третата снимка → `image3.jpg`
   - Четвъртата снимка → `image4.jpg`
   - Петата снимка → `image5.jpg`

## Резолюция и оптимизация:
- **Целева резолюция:** 1080x1920px (portrait)
- **Формат:** JPG
- **Качество:** 85-90%
- **Размер:** под 300KB

## След като запазиш снимките:
1. Изтрий този файл `INSTRUCTIONS.md`
2. Изтрий `image1.jpg` placeholder файла
3. Рестартирай dev сървъра
4. Ротацията ще работи автоматично

## Как работи:
- При всяко зареждане на сайта се избира случайна снимка
- Системата вече е имплементирана и работи
- Ще се вижда различна снимка при всеки рефреш
